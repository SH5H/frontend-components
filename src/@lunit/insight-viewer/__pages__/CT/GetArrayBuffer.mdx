# Get `ArrayBuffer`

CT 관련 이미지는 Python NumPy (`*.npy`) 형식으로 넘어오게 된다.

NumPy Data를 Bytes로 읽어들여 사용해야 하므로,

일단 `ArrayBuffer` 형태의 데이터를 얻을 수 있는 수단들이 필요하다.
