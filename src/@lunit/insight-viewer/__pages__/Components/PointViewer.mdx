# `<PointViewer>`

`<ContourViewer>`, `<CircleViewer>`와 마찬가지로 `Contour` 데이터를 화면에 그린다.

<Link to="/insight-viewer/contour">`Contour` 데이터를 `Point` Type으로 사용한다.</Link>

`<ContourViewer>` + `<ContourDrawer>` 처럼 분리되어 있지 않고, `<PointViewer>`에서 View 와 Draw를 모두 처리한다.
