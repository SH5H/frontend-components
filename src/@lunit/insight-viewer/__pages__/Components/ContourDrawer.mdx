# `<ContourDrawer>`

Annotation Tool, OPT 와 같이 사용자의 Contour Drawing이 필요한 경우 사용된다.

단순히 그리는 기능만 있고, 그려진 Contour 데이터들은 `<ContourViewer>`에서 처리해야 한다.
