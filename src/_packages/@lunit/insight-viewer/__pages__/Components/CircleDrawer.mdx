# Styling

`<ContourDrawer>`와 마찬가지로 CSS Variable을 통해서 색상을 변경할 수 있다.

`<ContourDrawer>`와 마찬가지로 `--contour-drawer-*`의 이름을 가진 항목들에 영향을 받는다.
