# `useImageStore()`

`*.dcm` 파일을 Prefetch 하기 위해 사용한다.

사용자가 다음에 볼 것이라 예상되는 `CornerstoneImage`를 미리 Prefetch 할 수 있다.

- 현재 1번 Image를 보고 있다면
- `useEffect(() => fetch(imageIds[2]), [imageIds])` 이와 같은 식으로 2번 이미지를 Prefetch 한다
